# msc_comp_path

🚀 Computational Pathology Course Setup Instructions

Please complete these steps before the course starts.

# Step 1: Install Git
- Windows: Download and install Git for Windows. Just use the default settings during installation.

- Mac: Open your Terminal and type git --version. If it’s not there, a window will pop up asking you to install "Command Line Developer Tools." Click Install.

# Step 2: Clone the Project
Open your Terminal (Mac) or Command Prompt (Windows) and run:

git clone https://github.com/coolpaul/msc_comp_path.git
cd msc_comp_path

# Step 3: Run the Auto-Setup
- Windows: Double-click setup_windows.bat.

- Mac: Right-click setup_mac.command -> Open -> Open anyway.

# Step 4: Launch the Course Environment
Once the setup finishes, run:

conda activate msc_comp_path
