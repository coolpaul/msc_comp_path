3.1.3 Dates

Parsing dates is often difficult due to difference in notation: 05/11/2026. In America, this is regarded as 11th May, whilst this is 5th November in UK. In similarity with numbers, it is recommended to use the ISO 8601 standard for dates. The most significant number is on the left.

3.5.2 Pandas

Pandas stores data in a dataframe. Every dataframe can have one or more variables. A variable can only contain one category (number, categorical, etc), but there can be many variables in a dataframe each of a different type.

3.5.4 Grammar of graphics

If data are tidy, building plots is usually straight forward using the grammar of graphics.

Familiarise yourself with the grammar of graphics to build plots Wickham (2010). A plot is a graphical representation of the data. It requires a minimum of three elements:

  • a tidy dataframe
  • aesthetic elements
  • geometric element(s)

The dataframe should be in tidy format to allow ggplot (R) or plotnine (Python) to build the plot. The dataframe contains variables that could be numerical or categorical. The geometric element describes what type of plot is required (histogram, regression plot, box and whisker plot) and is related to the aesthetic element. The aesthetic element describes how the data is mapped on the plot. For example, a histogram only requires a x-axis, a regression plot an x- and y-axis, a box and whisker plot a categorical x-axis and a numerical y-axis. Additionally, variables can be allocated to colour, fill, size, transparancy etc. python
